# Personal Portfolio Website

## Overview

This project is a Full-Stack Personal Portfolio Website developed to showcase my skills, projects, and certifications. The application integrates a frontend, backend, and database to demonstrate full-stack development concepts and deployment readiness.

## Features

* Responsive portfolio website
* Professional user interface built with HTML, CSS, and JavaScript
* Backend API built using Node.js and Express.js
* MongoDB database integration
* Dynamic project data retrieval through REST APIs
* Certifications and contact information display

## Technology Stack

### Frontend

* HTML5
* CSS3
* JavaScript

### Backend

* Node.js
* Express.js

### Database

* MongoDB

## Projects Included

### Task Manager

A web application that allows users to create, update, and manage daily tasks efficiently.

### Password Strength Checker

An application that evaluates password strength based on length, complexity, and security best practices.

### Port Scanner

A network utility that scans target systems to identify open ports and running services.
